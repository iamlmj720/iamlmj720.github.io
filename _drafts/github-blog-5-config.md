## 이번 편에서 할 것
- ✅ 결과물: 내 색깔이 입혀진 블로그

## 1. _config.yml 다루는 법
- 수정 후 **서버 재시작 필요**하다는 점 (자동 반영 안 됨)

## 2. 사이트 기본 정보
- title, description, url, logo/teaser

## 3. 작가 프로필 (사이드바)
- name, avatar, bio, location, 소셜 링크(GitHub/Instagram 등)

## 4. 스킨 & 디자인
- minimal_mistakes_skin (dark 등) 바꿔보기

## 5. About 페이지 꾸미기
- `_pages/about.md` 구성, 자기소개에 넣으면 좋은 것
