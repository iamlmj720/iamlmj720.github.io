> 📚 GitHub 블로그 만들기 시리즈 #2 · [목차 보기](#)

## 이번 편에서 할 것
- ✅ 결과물: 인터넷에 내 빈 블로그가 뜬다

## 1. 저장소 만들기
- `username.github.io` 형식으로 저장소 생성
- minimal-mistakes 가져오기 (fork / 템플릿 / gem 방식 중 택1)

## 2. 로컬 환경 설치
- Ruby, RubyGems, Bundler 설치 (macOS/Windows 차이 메모)
- `bundle install`로 의존성 설치

## 3. 로컬 미리보기
- `bundle exec jekyll serve` → `localhost:4000` 접속
- 자주 만나는 에러 & 해결 (버전 충돌 등)

## 4. GitHub Pages로 배포
- push → Settings의 Pages 설정 → 실제 주소 확인
