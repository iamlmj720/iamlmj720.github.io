> 📚 GitHub 블로그 만들기 시리즈 #3 · [목차 보기](#)

## 이번 편에서 할 것
- ✅ 결과물: 내 첫 글이 블로그에 올라간다

## 1. 글 파일 규칙
- `_posts/YYYY-MM-DD-제목.md` 형식
- 파일명/날짜가 URL과 정렬에 미치는 영향

## 2. front matter 항목별 설명
- title, categories, tags, date, toc, excerpt, header.teaser ...
- 각 항목이 화면 어디에 나타나는지

## 3. 마크다운 기본
- 제목/목록/인용/코드블록/링크
- 코드 안에 Liquid 쓸 때 `{% raw %}` 주의점

## 4. 이미지 넣기
- 이미지 폴더 구조, `relative_url` 사용법
